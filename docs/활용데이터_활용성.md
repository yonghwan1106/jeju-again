# 📊 활용데이터 (활용성)

## 1. 제주관광 빅데이터 서비스 플랫폼 (필수)

**출처**: data.ijto.or.kr (제주관광빅데이터 플랫폼)

### 1.1 입도·이동 통계 데이터

#### 제주 내국인 입도 통계 (2019-2025)
- 월별/분기별 내국인 관광객 입도 현황
- 방문 목적별 분류 (휴양및관람, 레저스포츠, 친지방문, 회의및업무, 교육여행, 기타)
- 여행 형태별 분류 (개별여행, 패키지, 부분패키지)
- **활용 방식**: 통계 페이지에서 실시간 입도객 현황 시각화, 관광약자 비율 추정
- **실제 데이터**: 2025년 8월 내국인 입도객 1,083,603명 (직접 다운로드)

#### 제주 외국인 입도 통계 (2019-2025)
- 국가별 외국인 관광객 입도 현황
- 제주 직항 vs 한국 경유 구분
- **활용 방식**: 국가별 관광객 분포 Top 5 차트, 월간 통계 분석
- **실제 데이터**: 2025년 8월 외국인 입도객 2,065,840명 (직접 다운로드)
  - 중국 38.2%, 일본 18.6%, 대만 10.4% 등

#### 지역별 관광객 현황 (2019-2025)
- 읍면동별 관광객 분포 데이터
- 지역별 관광객 증감률
- **활용 방식**: 지역별 관광객 분포 시각화 → 혼잡 지역 회피 알고리즘에 활용
- **주요 인사이트**: 제주시 동지역 35.2% 집중 → 동부/서부 지역 분산 전략 수립

#### 지역별 여행 키워드 (2021-2025)
- 통신사 기준 방문지역 순위
- 비짓제주 여행지 키워드
- **활용 방식**: 관광 트렌드 파악 및 추천 알고리즘 개선

### 1.2 생산·소비 통계 데이터

#### 월간 제주 소비 동향 (2013-2025)
- 업종별/지역별/출신지역별 소비금액 추이
- **활용 방식**: 통계 페이지의 "업종별 소비 금액 현황" 섹션에 표시
- **실제 데이터**:
  - 소매업 2,026억원 (+5.2%)
  - 음식점업 1,872억원 (+3.8%)
  - 숙박업 956억원 (-2.1%)

#### 업종별 소비분석 (2013-2025)
- 내/외국인 구분 매출액 및 증감률
- **활용 방식**: 지역 상생 로컬 상점 추천 시 소비 트렌드 반영

### 1.3 관광동향 보고서 데이터

#### 한라산 탐방객 통계 (2021-2025)
- **활용 방식**: 주요 관광지 혼잡도 예측 데이터로 활용

#### 제주 운항편 수 및 여객수 (2023-2025)
- **활용 방식**: 계절별/요일별 관광객 유입 패턴 분석

#### 기타 관광동향 데이터
- 여객선 이용객, 렌터카 가동률, 전세버스 가동률
- 제주 섬 속의 섬 이동객 (우도, 가파도, 마라도, 비양도, 추자도)
- **활용 방식**: 이동 패턴 분석 및 교통 혼잡도 예측

## 2. 타 기관 데이터 (선택)

### 2.1 한국관광공사 Tour API
- **제주 관광지 정보 (30개 POI)**
  - GPS 좌표, 주소, 카테고리
  - 접근성 정보 (유모차, 휠체어, 주차장, 화장실, 수유실 등)
  - **활용 방식**: AI 여행 코스 생성 시 실제 관광지 데이터로 활용
  - **출처**: 한국관광공사 공공데이터 포털
  - **구현 파일**: `data/real/jejuTouristSpots.json`

**주요 관광지 예시**:
- 한라산 국립공원, 성산일출봉, 만장굴, 협재해수욕장
- 우도, 비자림, 천지연폭포, 정방폭포
- 제주올레길, 산방산, 섭지코지 등

### 2.2 Naver Maps API
- **지도 시각화 및 경로 탐색**
  - 관광지 위치 표시 (마커)
  - 경로 계산 및 시각화 (폴리라인)
  - 거리 및 소요시간 계산
  - **활용 방식**: 생성된 여행 코스를 지도에 표시하고 경로 안내
  - **API**: Naver Maps JavaScript API v3
  - **구현**: `components/NaverMapView.tsx`

### 2.3 공공데이터 포털
- **관광약자 통계**
  - 65세 이상 시니어 비율: 45%
  - 영유아 동반 가족 비율: 41%
  - **활용 방식**: 관광약자 맞춤형 서비스 설계의 근거 데이터
  - **프로파일 설계**: ACTIVE_SENIOR, CARE_SENIOR, TODDLER_FAMILY, INFANT_FAMILY

## 3. 데이터 활용 프로세스

### 3.1 데이터 수집
```
├─ data.ijto.or.kr에서 CSV/Excel 직접 다운로드
│  ├─ 제주내국인_입도통계_202508.csv/xlsx
│  └─ 제주외국인_입도통계_202508.csv/xlsx
│
└─ 공공 API 연동
   ├─ 한국관광공사 Tour API (관광지 정보)
   └─ Naver Maps API (지도 및 경로)
```

### 3.2 데이터 처리
```
scripts/process-downloaded-data.ts
├─ CSV 파싱 (csv-parse 라이브러리)
├─ 데이터 정제 및 집계
│  ├─ 목적별 집계 (휴양및관람, 레저스포츠 등)
│  ├─ 국가별 집계 및 순위 계산
│  └─ 퍼센트 계산
└─ JSON 변환 및 저장
   └─ data/real/touristStatistics202508.json
```

### 3.3 데이터 통합
```
├─ 통계 페이지 (app/statistics/page.tsx)
│  ├─ 실시간 입도객 현황 표시
│  ├─ 지역별 관광객 분포 차트
│  ├─ 업종별 소비 금액 시각화
│  └─ 실제 다운로드 데이터 월간 상세 분석
│
├─ AI 코스 생성 (app/api/generate-itinerary/route.ts)
│  ├─ 혼잡도 데이터 기반 장소 필터링
│  ├─ 프로파일 맞춤 장소 선정
│  └─ 지역 분산 알고리즘 적용
│
└─ 데이터 센터 (app/data-center/page.tsx)
   └─ 실제 다운로드 페이지 링크 제공
```

### 3.4 사용자 제공
```
├─ 통계 대시보드
│  └─ /statistics - 실시간 관광 통계 시각화
│
├─ 데이터 센터
│  └─ /data-center - 데이터 카탈로그 및 다운로드 링크
│
└─ AI 맞춤형 여행 코스
   └─ / - 프로파일 기반 여행 코스 생성
```

## 4. 데이터 활용의 차별성

### ✅ 실제 다운로드한 공식 데이터 사용
- data.ijto.or.kr에서 직접 다운로드한 **2025년 8월 실제 입도 통계** 적용
- 검증 가능한 출처 명시 (CSV 파일 보관: `data/real/downloaded/`)
- **증빙 가능**: GitHub 리포지토리에 원본 CSV 파일 및 처리 스크립트 공개

### ✅ 데이터 기반 의사결정
```
혼잡도 데이터
  → 지역별 관광객 분포 분석
  → 제주시 동지역 35.2% 집중 파악
  → AI가 동부/서부 지역 우선 추천
  → 관광객 분산 효과

소비 통계
  → 지역별 소비 증가율 분석
  → 증가율 높은 지역의 로컬 상점 우선 추천
  → 지역 상생 효과

접근성 데이터
  → 유모차/휠체어 접근 가능 여부
  → 주차장/화장실/수유실 유무
  → 관광약자 맞춤 코스 생성
```

### ✅ 투명한 데이터 출처 공개
- 모든 통계에 **출처 명시** (제주관광협회, 한국공항공사 등)
- 데이터 센터 페이지에서 **원본 다운로드 링크 제공**
- **최종 갱신일** 명시: 2025-08-31

### ✅ 자동화된 데이터 처리 파이프라인
```typescript
// scripts/process-downloaded-data.ts
CSV 다운로드
  → 파싱 및 집계
  → JSON 변환
  → 웹 페이지 통합
  → 실시간 시각화
```
- 수동 작업 최소화
- 데이터 업데이트 시 스크립트 재실행으로 자동 반영
